## 🛒 Demo EC Store (Client Site)

マーケティングツールの動作を検証・シミュレーションするためのデモ用ECサイトです。

### 🎯 役割
- **高度な検索体験**: Elasticsearchを使用したミリ秒単位の集計・ハイライト表示。
- **行動データの収集**: `Tracker.tsx` によるリアルタイムなイベント送信。
- **インサイト連動型UI**: SaaS側の解析結果に応じた、動的なUIの出し分け。

### 🔍 検索・パーソナライズ仕様
- **リアルタイム検索**: デバウンス処理（500ms）とハイライト表示。
- **📍 インサイト解析 (Behavioral Scoring)**: 
    - **多角的スコアリング**: 「安い順」へのソート切替や、特定カテゴリの閲覧、特定の検索キーワード（シノニム）によってスコアを蓄積。
    - **UX配慮型配信**: インサイト合致時、検索の邪魔にならないよう **2.5秒の待機時間** を経てから右下にふわっとポップアップを表示。
    - **デザイン出し分け**: 判定されたユーザーには「INSIGHT MATCH」として、その人の意図（節約・ギフト・急ぎ）に沿った内容を優先表示。

### 🔌 マーケティングツールとの連携
1. **行動トラッキング**: `POST http://localhost:3001/api/v1/track` へイベントを送信。
2. **インサイト同期**: SaaS側の解析結果（スコア）を受け取り、最新のセグメントに基づいた接客を実行。
